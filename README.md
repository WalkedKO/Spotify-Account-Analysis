## The data
I used my top listened tracks and top listened artists data using Spotify API. Additionally, I scrapped the music genres from wikipedia.
## Analysis
I analysed my listening history based on three periods: last month, last 6 months, last year.
## How to use?
Just open and see the SpotifyAccountAnalysis.ipynb. If you want to load files, there are json files in data folders which are downloaded from SpotfiyAPI and csv files which contain cleaned and structured data generated by first parts of the notebook.
If you want to generate your own data, you have to generate a token from Spotify API for top tracks/artists, and paste it into the token variable in the notebook
